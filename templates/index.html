Serious Chicken

A small static site generator made for developing large sites.

All kidding a side, this project was trying to hit a few points:

  • Instead of building a full separate application, use libraries that make use of this tech and throw them together to make a small static site generator
  • Make pages and templates easily customizable, use markdown with metadata
  • Keep both develop and build fast and smooth. While developing, it is just your average dynamic python site, then you use the build command later on to generate the static pages

You can take a look at the source here.

This site itself was build with Serious Chicken.
